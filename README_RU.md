# AVZ Game Gears Test — web-презентация v01

Это новая версия подачи тестового: не PPTX/Google Slides, а статическая web-презентация.

## Почему web-формат

- Ссылка открывает презентацию сразу.
- Работает на ПК, MacBook и мобильном.
- Интерактивный калькулятор экипировки НЛО работает прямо внутри слайда.
- Proof-of-work распределения игроков по турнирным комнатам работает прямо внутри слайда.
- Не требуется запускать отдельные приложения проверяющему.

## Как запустить локально

Открой `index.html` в браузере.

Лучше через локальный сервер:

```bash
python3 -m http.server 4173
```

После этого открыть:

```text
http://127.0.0.1:4173
```

## Как опубликовать по ссылке

Самый простой вариант:

1. Залить содержимое папки `avz_web_presentation_v01` на GitHub.
2. Включить GitHub Pages для ветки/папки.
3. Полученную ссылку отправить проверяющему.

Для репозитория `avz-game-gears-test` рекомендуется загрузить:

- `index.html`
- папку `assets/`
- папку `docs/`
- `README_RU.md`

Файл `index_single_file.html` — backup-версия, где картинки вшиты прямо в HTML. Она удобна для проверки одним файлом, но тяжелее для редактирования.

Альтернативы:

- Netlify Drop: перетащить папку в интерфейс Netlify.
- Vercel: импортировать папку как static project.

## Google Sheet

В папке `docs/` лежит Apps Script для создания Google Sheet-версии калькулятора:

```text
docs/Code.gs
```

В отдельном файле `avz_ufo_equipment_calculator_google_sheet_seed_v01.xlsx` лежит заготовка таблицы с листами:

- `Настройка`
- `Балансовая таблица`
- `Визуализация данных`

XLSX-файл можно использовать как офлайн-заготовку/референс, но для финальной ссылки лучше создать настоящую Google Sheet и заполнить ее через Apps Script.

### Как создать Google Sheet через Apps Script

1. Открыть Google Drive.
2. Создать новую таблицу: `Создать -> Google Таблицы`.
3. Назвать файл, например: `AVZ UFO Equipment Calculator`.
4. В верхнем меню таблицы открыть: `Расширения -> Apps Script`.
5. В открывшемся редакторе Apps Script удалить стандартный код из файла `Code.gs`.
6. Открыть локальный файл проекта `docs/Code.gs`.
7. Скопировать весь код из `docs/Code.gs` и вставить его в редактор Apps Script.
8. Нажать `Save project` / иконку сохранения.
9. Вернуться во вкладку Google Sheet и обновить страницу.
10. В верхнем меню таблицы должен появиться новый пункт `UFO Calculator`.
11. Нажать `UFO Calculator -> Создать демо-таблицу`.
12. При первом запуске Google может запросить разрешения:
    - выбрать свой Google-аккаунт;
    - нажать `Advanced` / `Дополнительно`, если Google показывает предупреждение;
    - выбрать переход к проекту;
    - нажать `Allow` / `Разрешить`.
13. После выполнения скрипта в таблице должны появиться/обновиться листы:
    - `Настройка`
    - `Балансовая таблица`
    - `Визуализация данных`
14. Открыть лист `Визуализация данных`.
15. Проверить, что в ячейках `B11:B13` работают выпадающие списки предметов.
16. Изменить выбранные предметы и убедиться, что блок итоговых характеристик обновляется.

Если меню `UFO Calculator` не появилось:

1. Убедиться, что код вставлен именно через `Расширения -> Apps Script` из нужной Google Sheet.
2. В Apps Script нажать `Save project`.
3. Вернуться в Google Sheet и обновить страницу.
4. Если меню все еще не появилось, в Apps Script выбрать функцию `onOpen` и нажать `Run`, затем снова обновить Google Sheet.
5. Если нужно запустить без меню, в Apps Script выбрать функцию `setupUfoCalculator` и нажать `Run`.

### Как открыть доступ по ссылке

1. В Google Sheet нажать `Share` / `Поделиться`.
2. В блоке доступа выбрать вариант `Anyone with the link` / `Все, у кого есть ссылка`.
3. Роль можно оставить `Viewer` / `Читатель`, если проверяющему нужно только посмотреть таблицу.
4. Если проверяющий должен менять выпадающие списки, выбрать `Editor` / `Редактор`.
5. Скопировать ссылку на Google Sheet.

### Как вставить ссылку в web-презентацию

В `index.html` найти:

```js
googleSheetUrl: "#google-sheet-link-pending"
```

и заменить `#google-sheet-link-pending` на реальную ссылку Google Sheet.
