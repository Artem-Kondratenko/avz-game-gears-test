# AVZ Game Gears Test — web-презентация v01

Это новая версия подачи тестового: не PPTX/Google Slides, а статическая web-презентация.

## Почему web-формат

- Ссылка открывает презентацию сразу.
- Работает на ПК, MacBook и мобильном.
- Интерактивный калькулятор экипировки НЛО работает прямо внутри слайда.
- Proof-of-work распределения игроков по турнирным комнатам работает прямо внутри слайда.
- Не требуется запускать отдельные приложения проверяющему.

## Как запустить локально

Открой `index.html` в браузере.

Лучше через локальный сервер:

```bash
python3 -m http.server 4173
```

После этого открыть:

```text
http://127.0.0.1:4173
```

## Как опубликовать по ссылке

Самый простой вариант:

1. Залить содержимое папки `avz_web_presentation_v01` на GitHub.
2. Включить GitHub Pages для ветки/папки.
3. Полученную ссылку отправить проверяющему.

Альтернативы:

- Netlify Drop: перетащить папку в интерфейс Netlify.
- Vercel: импортировать папку как static project.

## Google Sheet

В папке `docs/` лежит Apps Script для создания Google Sheet-версии калькулятора.

В отдельном файле `avz_ufo_equipment_calculator_google_sheet_seed_v01.xlsx` лежит заготовка таблицы с листами:

- `Настройка`
- `Балансовая таблица`
- `Визуализация данных`

Чтобы вставить ссылку на Google Sheet в web-презентацию:

1. Создать Google Sheet.
2. Вставить Apps Script из `docs/Code.gs`.
3. Запустить пункт меню `UFO Calculator -> Создать демо-таблицу`.
4. Открыть доступ по ссылке.
5. В `index.html` заменить:

```js
googleSheetUrl: "#google-sheet-link-pending"
```

на реальную ссылку.
